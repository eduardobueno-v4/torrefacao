# B2B Coffee Insights — Análise de Mercado

Dashboard interativo de análise do mercado de Torrefação B2B e Café White Label.

## Tecnologias
- HTML5 puro
- Tailwind CSS (CDN)
- Chart.js (CDN)
- JavaScript vanilla

## Como rodar localmente
Abra o arquivo `index.html` diretamente no navegador.

## Deploy online

### Opção 1 — Netlify Drop (mais fácil, sem conta necessária)
1. Acesse [netlify.com/drop](https://app.netlify.com/drop)
2. Arraste e solte a pasta `site brenin` inteira no site
3. Pronto! Você receberá um link público imediatamente.

### Opção 2 — Netlify via GitHub
1. Suba esta pasta para um repositório no GitHub
2. Acesse [netlify.com](https://netlify.com) e conecte seu repositório
3. Branch: `main`, pasta de publicação: `.`

### Opção 3 — Vercel via GitHub
1. Suba esta pasta para um repositório no GitHub
2. Acesse [vercel.com](https://vercel.com) e importe o repositório
3. Framework: `Other`, sem build command necessário
